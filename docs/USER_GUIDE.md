## Getting Started

### Creating an Account

1. Navigate to the registration page
2. Fill in your details:
   - Email address
   - Password (minimum 6 characters)
   - Full name
   - Phone number (optional)
   - Company (optional)
3. Select your role (default: Attendee)
4. Click "Register"

### Logging In

1. Navigate to the login page
2. Enter your email and password
3. Click "Login"
4. You'll be redirected to the dashboard

## Attendee Guide

### Browsing Events

1. Navigate to "Events" from the main menu
2. Browse available events
3. Use filters to search:
   - Search by title/description
   - Filter by date range
   - Filter by location

### Registering for an Event

1. Click on an event to view details
2. Review event information:
   - Date and time
   - Location
   - Available capacity
   - Description
3. Click "Register" button
4. Confirm your registration
5. You'll receive a confirmation with your ticket

### Viewing Your Tickets

1. Navigate to "My Tickets"
2. View all your registered events
3. Click on a ticket to see details:
   - Event information
   - QR code
   - Ticket code
   - Registration status

### Using Your QR Code

1. Open your ticket
2. Display the QR code on your device
3. Present it to the organizer at check-in
4. Wait for confirmation

**Tips:**
- Increase screen brightness for better scanning
- Keep your device charged
- Take a screenshot as backup

### Canceling a Registration

1. Go to "My Tickets"
2. Select the ticket you want to cancel
3. Click "Cancel Registration"
4. Confirm cancellation
5. Your spot will be freed for others

## Organizer Guide

### Creating an Event

1. Navigate to "Events" → "Create Event"
2. Fill in event details:
   - **Title**: Event name
   - **Description**: Detailed information
   - **Date & Time**: When the event occurs
   - **Location**: Where it takes place
   - **Capacity**: Maximum attendees
   - **Image**: Event banner (optional)
3. Click "Create Event"
4. Your event is now live!

### Managing Events

#### Editing an Event

1. Go to "My Events"
2. Click on the event you want to edit
3. Click "Edit" button
4. Update information
5. Save changes

#### Deleting an Event

1. Go to "My Events"
2. Click on the event
3. Click "Delete" button
4. Confirm deletion
5. All registrations will be canceled

### Checking In Attendees

#### Using QR Scanner

1. Navigate to "Check-in" → "Scanner"
2. Select the event
3. Allow camera access
4. Point camera at attendee's QR code
5. Wait for confirmation:
   - ✅ Green: Successful check-in
   - ❌ Red: Invalid/duplicate ticket
   - ⚠️ Yellow: Already checked in

#### Manual Check-in

If QR scanning fails:

1. Navigate to "Check-in" → "Manual"
2. Search for attendee by:
   - Name
   - Email
   - Ticket code
3. Select the attendee
4. Click "Check In"
5. Confirm action

### Viewing Event Statistics

1. Go to your event details
2. Click "Statistics" tab
3. View metrics:
   - Total registrations
   - Check-in count
   - Check-in rate
   - Recent check-ins
   - Attendance trends

### Exporting Attendee Lists

1. Open event details
2. Click "Export" button
3. Choose format:
   - CSV (Excel compatible)
   - PDF (printable)
4. Download file

## Admin Guide

### Managing Users

#### Viewing All Users

1. Navigate to "Admin" → "Users"
2. View user list with:
   - Name and email
   - Role
   - Registration date
   - Status

#### Changing User Roles

1. Go to "Admin" → "Users"
2. Find the user
3. Click "Edit Role"
4. Select new role:
   - Admin
   - Organizer
   - Attendee
5. Confirm change

**Note**: Be careful when assigning Admin role!

#### Deleting Users

1. Go to "Admin" → "Users"
2. Find the user
3. Click "Delete" button
4. Confirm deletion
5. All user data will be removed

### System Statistics

1. Navigate to "Admin" → "Dashboard"
2. View system-wide metrics:
   - Total users
   - Total events
   - Total registrations
   - Total check-ins
   - Users by role
   - Recent activity

### Analytics

1. Go to "Admin" → "Analytics"
2. View charts and graphs:
   - User growth over time
   - Event popularity
   - Registration trends
   - Check-in rates
   - Peak usage times

### Exporting Data

1. Navigate to "Admin" → "Events"

3. Choose format (CSV/Excel)
4. Download file

#### Export User Data

1. Go to "Admin" → "Users"

4. Download file

## Troubleshooting

### Login Issues

**Problem**: Can't log in

**Solutions**:
- Verify email and password
- Check for typos
- Reset password if forgotten
- Clear browser cache
- Try different browser

### Registration Issues

**Problem**: Can't register for event

**Possible causes**:
- Event is full
- Already registered
- Event has passed
- Network connection issue

**Solutions**:
- Check event capacity
- Verify you're not already registered
- Check event date
- Refresh the page
- Contact organizer

### QR Code Issues

**Problem**: QR code won't scan

**Solutions**:
- Increase screen brightness
- Clean camera lens
- Hold device steady
- Ensure good lighting
- Try manual check-in
- Use backup ticket code

### Missing Tickets

**Problem**: Can't find my tickets

**Solutions**:
- Check "My Tickets" page
- Verify you're logged in
- Check email for confirmation
- Contact event organizer
- Check spam folder

### Performance Issues

**Problem**: App is slow

**Solutions**:
- Clear browser cache
- Close other tabs
- Check internet connection
- Try different browser
- Restart device

### Email Notifications

**Problem**: Not receiving emails

**Solutions**:
- Check spam/junk folder
- Verify email address
- Add sender to contacts
- Check email settings
- Contact support

## Best Practices

### For Attendees

1. **Register Early**: Events can fill up quickly
2. **Save QR Code**: Take screenshot as backup
3. **Arrive Early**: Allow time for check-in
4. **Keep Device Charged**: Ensure battery for QR code
5. **Update Profile**: Keep contact info current

### For Organizers

1. **Clear Descriptions**: Provide detailed event info
2. **Set Realistic Capacity**: Don't overbook
3. **Test Scanner**: Practice before event day
4. **Have Backup Plan**: Manual check-in ready
5. **Monitor Statistics**: Track attendance in real-time
6. **Communicate**: Send reminders to attendees

### For Admins

1. **Regular Backups**: Export data regularly
2. **Monitor Activity**: Check for suspicious behavior
3. **Manage Roles**: Review user permissions
4. **System Health**: Monitor performance
5. **User Support**: Respond to issues promptly
